Working with Positions

Sticky

This is an example of an element with a sticky position. Its position is based on the user's scroll position.

The sticky position is used to keep an element within its block while you scroll through the webpage. You can achive this by using the position: sticky; rule in CSS or position: -webkit-sticky for Safari. Then you specifiy where you want the element to stay in this can be done by using the top, bottom, left, or right declaration. Once you done that you would have to set where in the page you want it to stay an example is with the top: 0; which keeps the block at the top of the webpage.

Relative

This is an example of an element with a relative position. It can move around a little, depending on the viewport, but it is in a position relative to itself.

I used the 75px for the top property and this changed the box by moving it up more than the original. Then I used the 20px for the left property and this changed the box by moving it to the left more than the original.

Fixed

This is an example of an element with a fixed position. Its position is relative to the viewport.

I used 60px for the top property and this changed the box by moving it down slightly from the original. Then I used 30px for the right property and this changed the box by moving it to the right more than original.

Absolute

This is an example of an element with an absolute position. It is similar to the fixed position; however, it is positioned relative to the closest parent element.

I used 30px for the bottom property and this changed the box by moving it 30px lower on the page. Then I used 45px for the right property and this changed the box by moving it over to the right by 45px.
